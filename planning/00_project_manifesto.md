# Project Manifesto

## Aim

The project is a comprehensive multi-volume monograph on quantum field theory
for advanced PhD students, researchers, and mathematically capable AI agents.
It begins from the logical order of Xi Yin's QFT notes and expands that order
into a deeper, broader, and more rigorous work.

The monograph should teach QFT as a coherent construction of local quantum
physics. It should make each object visible as an object: its data, symbols,
domain, hypotheses, covariance laws, positivity conditions, limiting
procedures, and relation to earlier constructions.

## Foundation Stance

There is no currently known axiom system that is both mathematically complete
and physically adequate for all aspects of QFT needed in this monograph.
Wightman QFT, Osterwalder--Schrader reconstruction, algebraic QFT,
perturbative AQFT, constructive QFT, factorization-algebraic approaches,
functorial/topological frameworks, and Kontsevich--Segal-type ideas each
capture important structures and each has a specific domain.

The monograph therefore uses a modular foundation. Each construction states its
own data and hypotheses. A theorem is valid only in the framework where its
hypotheses have been stated. A comparison with an axiomatic framework is made
when it clarifies a domain, a theorem, a limitation, or an open mathematical
problem.

## Spine

The logical order follows Xi Yin's QFT notes. This is a structural constraint,
not a loose inspiration.

- Local quantum physics, symmetry, locality, and fields come first.
- Kallen--Lehmann spectral representation appears early.
- Path integrals and perturbation theory are introduced first for Green
  functions.
- The S-matrix is introduced first as a nonperturbative comparison of
  asymptotic particle states under appropriate hypotheses.
- LSZ follows the nonperturbative scattering construction.
- Feynman diagrams for S-matrix elements appear only after LSZ has supplied
  that interpretation.

The manuscript must not reorder the subject according to common textbook
habit, historical development, or computational convenience.

## Non-Negotiable Standards

- Define symbols before load-bearing use.
- State every framework before using claims that depend on it.
- Use genuine definitions, not simplified or informal substitutes.
- Distinguish primitive data from constructed objects.
- Distinguish exact, regulated, perturbative, formal, heuristic,
  conjectural, and open statements.
- State the order and meaning of limits.
- Treat path integrals by their actual mathematical status.
- Treat gauge symmetry as redundancy unless a physical symmetry has been
  separately defined.
- Treat particles as spectral or asymptotic structures under hypotheses, not as
  the primitive content of QFT.
- Treat renormalization as scale-dependent effective description and continuum
  structure, with counterterm methods as one coordinate technology.
- Treat figures as mathematical content.
- Keep misconception correction in remarks, footnotes, or appendices unless
  the correction is itself a precise construction in the main argument.

## Expository Rhythm

A mature section should normally proceed in this order:

1. framework;
2. data and symbols;
3. definition or construction;
4. assumptions;
5. derived consequences;
6. example or test case;
7. comparison, limitation, or open point if useful.

The main text should explain what the theory is, what each object is, and how
the objects are related. It should not rely on rhetorical contrasts, textbook
rebukes, or slogans as organizing devices.
