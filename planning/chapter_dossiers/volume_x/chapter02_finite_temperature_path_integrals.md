# Volume X, Chapter 2 Dossier: Finite-Temperature Path Integrals

  hydrodynamic variables.

- Gibbs trace and partition function.

## Claim Ledger

1. The trace imposes periodicity for bosonic configuration variables.
2. Fermionic coherent-state traces impose antiperiodicity for the thermal
   partition function and periodicity for the graded trace.
3. Matsubara correlators determine real-time response only with additional
   analytic growth or spectral input.
4. Chemical potentials are background holonomies along the Euclidean thermal
   circle.
