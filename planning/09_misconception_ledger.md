# Misconception Ledger

This ledger records common misleading presentations only when they affect the
monograph. It is not a polemical scorecard. The main text should present the
correct object first; misconception notes should usually appear as remarks,
footnotes, appendices, or planning-only warnings.

## Entry Standard

Add an entry only if:

- the misleading presentation is common enough to matter;
- it leads to incorrect reasoning;
- the monograph has a precise replacement statement;
- the correction has a chapter location;
- the treatment will not distort the main exposition.

## Treatment Options

- no explicit final mention, but avoid the error;
- footnote after the correct definition;
- remark after the correct theorem or construction;
- appendix comparison;
- planning-only note.

## Initial Entries

| Misleading presentation | Replacement statement | Treatment |
| --- | --- | --- |
| QFT is obtained by quantizing classical fields. | Some QFTs are constructed from classical Lagrangian data; the working object in the opening volume is local quantum data with states, observables, symmetry, spectrum, locality, and fields as distributional coordinates. | Remark only if useful. |
| A point field is an operator at a point. | A point field is an operator-valued distribution; smeared fields are the operator objects. | Main definition. |
| A path integral is an integral over all fields. | A path integral expression must be classified as finite-dimensional, regulated, Gaussian, constructive, perturbative formal, BRST/BV, or heuristic notation. | Main definition. |
| Wick rotation is a substitution. | Wick rotation is justified by analytic, spectral, reflection-positivity, or contour-deformation input. | Main derivation. |
| Gauge symmetry is an ordinary physical symmetry. | Gauge transformations are redundancies; physical symmetries act on gauge-invariant data. | Main definition and later remarks. |
| Renormalization is subtracting infinities. | Renormalization is scale-dependent effective description and continuum-limit structure; subtraction schemes are coordinate methods. | Main construction. |
| Particles are the primitive content of QFT. | Particles are spectral or asymptotic structures under stated hypotheses. | Main spectral and scattering chapters. |
| Feynman diagrams define the S-matrix. | Diagrams organize perturbative expansions; S-matrix interpretation requires asymptotic states and LSZ. | Main ordering rule. |
| Euclidean and Lorentzian QFT are automatically equivalent. | Reconstruction requires hypotheses such as reflection positivity and suitable analytic properties. | Framework comparison. |
| Axiomatic QFT supplies a final universal definition. | Existing frameworks have domains and unresolved comparison problems; the monograph uses claim-local frameworks. | Framework volume and planning. |
