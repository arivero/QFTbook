# Open Issue Development Map, 2026-05-25

## Scope

This file records the current full-open-issue review after the decision to
stop treating GitHub issues as an old-to-new queue.  It groups the open issues
by manuscript development track and records which new compiled volumes carry
the relevant work.

## Closure Rule

An issue is ready to close only after the monograph contains the substantive
development needed for the issue, the affected manuscript builds, local audits
pass, the change is pushed to `origin/main`, and the GitHub issue receives a
comment identifying the manuscript location and verification performed.
