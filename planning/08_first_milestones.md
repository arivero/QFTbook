# First Milestones

## Milestone 0: Planning Constitution

Rewrite the planning layer so that it enforces:

- logical order from the source notes;
- modular foundations;
- definition integrity;
- symbol discipline;
- source-first drafting;
- no slogan-based main exposition;
- only audited chapters in compiled TeX.

This milestone precedes further chapter expansion.

## Milestone 1: Reference Shelf

Build a reliable reference shelf for rigorous theorem boundaries:

- Wightman and reconstruction;
- Osterwalder--Schrader reconstruction;
- algebraic QFT;
- perturbative AQFT;
- Haag--Ruelle scattering;
- representation theory of the Poincare group;
- renormalization group;
- conformal bootstrap;
- gauge theory, BRST/BV, anomalies, and EFT.

Each downloaded reference needs a local record of its role. External
references support theorem status and framework comparison; they do not replace
the source spine.

## Milestone 2: Opening Chapters Re-Audit

Before adding more chapters, re-audit the compiled opening chapters for:

- all symbols defined;
- no informal definitions;
- no main-text negative framing;
- axiomatic frameworks stated only as comparison lenses;
- local field definitions treated distributionally;
- local algebra language stated with its domain and status.

## Milestone 3: Chapter 3 Dossier

Prepare the dossier for "Local Fields, Covariance, And Microcausality":

- source anchors in the first QFT sequence;
- definition of test-function spaces;
- operator-valued distributions;
- smeared fields;
- covariance laws;
- microcausality as a distributional statement;
- relation between field coordinates and local algebras;
- external references for Wightman-style field assumptions.

## Milestone 4: Chapter 3 Draft And Build

Draft Chapter 3 only after the dossier passes. Include it in the compiled
manuscript only after audit and build.

## Milestone 5: Continue The Spine

Proceed one chapter at a time:

1. dossier;
2. draft;
3. audit;
4. build;
5. source and claim report.

No unrevised scaffold chapter should enter the compiled TeX.
