# Issue 383: Anharmonic Wick Count

GitHub issue #383 flagged that the explanation of the factor
\(\langle q(\tau)^4\rangle_0=3G_0(\tau,\tau)^2\) could be read as the labeled
\(4!\) assignment count for a quartic vertex rather than the pair-partition
count for four coincident variables.

Edits made:

- Replaced the ambiguous sentence with the explicit statement that
  \(3=(4-1)!!\) counts pairings of four identical fields at the same Euclidean
  time among themselves.
- Added the distinction that the \(4!\) count applies when four labeled vertex
  half-edges are assigned to four labeled insertions, which is a different
  graph-combinatorics problem.
- Updated the chapter dossier claim ledger and certification notes.

This keeps the first-order vacuum correction tied to ordinary Gaussian Wick
pair partitions and prevents it from being confused with later diagram
symmetry-factor counts.
