# Issue #321 Audit: Curved-Background CFT Source Brackets

## Scope

- GitHub issue: `#321`, asking for the meaning of
  \(\langle\cdots\rangle_{g,\ast}\) in the opening CFT source-functional
  definition.
- Manuscript file:
  `monograph/tex/volumes/volume_iii/chapter01_fixed_points_and_conformal_data.tex`.

## Resolution

- The source-bracket definition now states explicitly that
  \(\langle\cdots\rangle_{g,\ast}\) means evaluation by the fixed-point
  curved-background partition/source functional
  \(Z_\ast[g,J]=\exp\{-W_\ast[g,J]\}\).
- The text explicitly says that this bracket is not, by notation alone, a
  claim that a literal Borel measure or an unregulated continuum path integral
  has been constructed.
- The construction-type classification remains:
  constructive Euclidean measure/generalized random field, lattice or
  Wilsonian continuum limit of regulated source functionals, abstract CFT
  generating functional for Euclidean Schwinger distributions with contact
  extensions, or perturbative formal source series.
- The text now adds the missing curved-background caveat: if a CFT
  presentation supplies only flat separated-point correlators, then
  \(Z_\ast[g,J]\), including its contact-term and Weyl-anomaly extension, is an
  additional datum or theorem before metric derivatives and curved-background
  Ward identities are used.
