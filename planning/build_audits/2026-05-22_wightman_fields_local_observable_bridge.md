# Wightman Field Coordinates and Local Observable Content

Date: 2026-05-22

Changes:
- Added a section separating Wightman field coordinates from local observable algebras.
- Defined the local unbounded domain algebra
  \(\mathfrak P_\Phi(\mathcal O)\) generated by smeared fields on the common invariant domain \(\mathcal D\).
- Stated explicitly that \(\mathfrak P_\Phi(\mathcal O)\) is not a \(C^*\)-algebra, and that a represented local von Neumann algebra requires additional essential self-adjointness, closure, bounded functional calculus, and localization hypotheses.
- Added the affiliation formulation for unbounded fields and local algebras, including the spectral-projection criterion for affiliated self-adjoint operators.
- Added Figure 2.2 to show the passage
  field coordinates -> domain algebra -> affiliation/bounded functional calculus -> local observable algebra,
  together with the separate equivalence question for another field coordinatization.
- Tightened the Wightman reconstruction theorem boundary: reconstruction is unique for the specified hierarchy and field labels, but does not by itself identify different field coordinatizations whose local observable algebras agree only after further closure, affiliation, or quotient constructions.
- Improved Figure 2.3 so the reconstruction diagram no longer has a label collision.
