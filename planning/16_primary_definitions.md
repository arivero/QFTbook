# Primary Definition Home Index

This file identifies the canonical home of load-bearing definitions and
theorems. Secondary appearances should refer back to the primary home rather
than silently redefining the object. The index is selective: it tracks concepts
whose duplicate definitions would create convention or logical ambiguity.

## Maintenance Rule

Before adding a duplicate definition of a load-bearing object, decide whether
the new passage is a primary home, a secondary explanation, or an application.
Secondary explanations must point back to the primary home. A new primary home
requires updating this file in the same commit.
