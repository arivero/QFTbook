# Deprecated Draft Material

This directory stores draft material that is not part of the compiled
monograph.  Files here are preserved for possible later reconstruction, but
they should not be treated as audited manuscript chapters.

The current policy is to keep the core QFT monograph focused on local quantum
fields, spectral theory, path integrals for Green functions, nonperturbative
scattering and LSZ, renormalization, gauge theory, anomalies, Wilsonian EFT,
and only the core conformal material needed for RG fixed points and local
operator algebra.
